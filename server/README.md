# Server

This sub-repo contains the back-end code for our project for CS3200.

## Set up

Create a `.env` file with the following flags:

```
SERVER_HOST= backend server host
SERVER_PORT= backend server port

DB_HOST= MySQL server host
DB_PORT= MySQL server port
DB_USER= MySQL server username
DB_PASSWORD= MySQL server password
DB_DATABASE= MySQL server schema
```

Run `npm start`.
